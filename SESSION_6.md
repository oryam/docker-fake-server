## Git

Créer un fork du project existant, puis clonner votre projet en local.
```
git clone https://github.com/<your_repo_name>/docker-fake-server.git
```

Découvrir le panneau et la vue git dans vscode. Voir l'historique des commits...

## Gemini et aide IA

Dans un navigateur, aller sur le site de Google Gemini et demander à générer un code simple HTML/CSS/Javascript.
Exemple de prompt :
```
generate html/js/css code to create a simple javascript app in a single html file, to manage TODOList locally (localstorage) with DnD support and responsive design
```

Essayer d'ajouter un bouton permettant de supprimer toutes les tâches.
